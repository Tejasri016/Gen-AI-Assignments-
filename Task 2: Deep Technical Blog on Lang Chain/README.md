# 🚀 LangChain Deep Dive: Building Modular LLM Applications

## 📌 Overview

This project demonstrates how to build modular and intelligent AI applications using **LangChain**. It combines both **theoretical understanding** and **practical implementation** of key concepts like prompt engineering, chains, memory, and agents.

---

## 📝 Blog Documentation

A detailed explanation of the project is provided in the blog document:

📄 **Blog link :** https://medium.com/@tejasrisomarouthu16/langchain-deep-dive-building-modular-llm-applications-with-python-69a37cbedf97

The blog covers:

* Introduction to LangChain
* Core components (LLMs, Prompts, Chains, Memory, Agents)
* Architecture and workflow
* Real-world use cases
* Advantages and limitations

---

## 💻 Implementation (Notebook)

The practical implementation is provided in the Jupyter Notebook:

📓 **Notebook File:** `LangChain-deep.ipynb` 
 
This includes:

* Prompt Templates for dynamic input
* Chain creation and execution
* Memory for context-aware conversations
* Document loading and retrieval
* Vector store basics

---

## 🔄 LLM Usage Note

The blog demonstrates usage with **OpenAI (ChatOpenAI)**.

However, due to API and billing constraints, the notebook uses a **local HuggingFace model** for execution.

👉 This shows that LangChain supports multiple LLM providers while maintaining the same workflow and architecture.

---

## ⚙️ How to Run

1. Install dependencies:

```bash
pip install langchain langchain-community transformers faiss-cpu
```

2. Open the notebook:

```bash
jupyter notebook Task1-prompt-engineering.ipynb
```

3. Run all cells step by step.

* Complex workflows can be harder to debug
